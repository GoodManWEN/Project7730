# Project7730

##### PART.A
使用聚簇化逻辑，基于KV数据库后端简化寻址和提取操作的用于处理特定时序任务的高性能数据抽象层。

##### PART.B
基于大规模数据的神经网络分析工程（待完善）

## 特性
- 为大规模数据量下高压力读取任务专门优化
- 更高储存效率、更低资源占用
- 高可用、通信安全特性
- 水平扩展性良好，支持容器化部署
